# 🎮 Bingo Seven Pro

Telegram Bot + Mini App + Bingo Engine

## Features
- Telegram bot commands (/startgame /next /state)
- Bingo number generator
- Mini Web App UI
- Ready for Render + Vercel

## Setup

1. npm install
2. create .env file
3. add BOT_TOKEN
4. npm start

## Deploy

- Bot → Render (Worker)
- Mini App → Vercel / Render Static Site
